# Foobar

Flight Booking System (Backend)
## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
 npm install node
 npm install express
 npm install pg 
 npm install sequalize
```

## Usage
##DB migration to ORM
```
  npx sequelize-cli db:migrate
````
````
##Start the Project: 
````
  node dev start 
````
